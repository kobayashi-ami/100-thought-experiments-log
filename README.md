# 100の思考実験 読書ログ
**ジュリアン・バジーニ著**  
開始: 2026-04-14

---

## 使い方

### ログを書く

```zsh
cd ~/reading/100-thought-experiments

# 新しいログを作る（番号 + タイトル）
./log.sh 1 "食われたがっている豚"

# 番号だけでもOK
./log.sh 7

# 既存ログを開く
./log.sh 3
```

### 進捗を確認する

```zsh
./stats.sh
```

### Claudeに手伝ってもらう

思考実験を読んで考えたことを話しかけるだけでOK。  
ログファイルに直接書いてもらえる。

---

## ファイル構成

```
100-thought-experiments/
├── README.md        ← これ
├── progress.md      ← 全100実験の進捗一覧
├── TEMPLATE.md      ← ログのテンプレート
├── log.sh           ← 新規ログ作成スクリプト
├── stats.sh         ← 統計表示スクリプト
└── logs/
    ├── 001.md       ← 各実験のログ
    ├── 007.md
    └── ...
```

---

## ログの書き方（自由でいい）

- 要約しなくていい。**引っかかったこと**を書く
- 答えを出さなくていい。**問いを深める**ための記録
- 数行でも、長文でも OK
